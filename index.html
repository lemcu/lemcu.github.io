EFM32ZG110 Quick Start Board

Easily prototype EFM32 Zero Geckos with this tiny quick start board. Available in a 24 pin dual inline breadboard compatible form. It is easy to program the microcontroller using the builtin serial bootloader. No external programmer is required.
